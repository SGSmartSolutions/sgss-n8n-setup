n8n Installer – Konfiguration

Trage hier deine Domain und Zugangsdaten ein. Danach erhältst du den Installationsbefehl für deinen Server.

A-Record deiner Domain muss auf die Server-IP zeigen.
Wird für SSL-Zertifikate verwendet (Let's Encrypt).
Wird abgefragt, bevor du n8n im Browser aufrufst.
Mindestens 32 Zeichen. Wird für Verschlüsselung sensibler Daten in n8n verwendet.
